WHAT IS BLOCKING?

[Image content]

IM Elements are ‘highly reactive’

Bond into blocks

Each Element may block with 2 others

Rules for blocking:

Extroverted + Introverted

Rational + Irrational

Static + Static OR Dynamic + Dynamic
